# 🌸 自律的迪莫 — 习惯打卡助手

一个可爱又实用的桌面习惯打卡应用。  
迪莫小宠物会陪伴你每日打卡，根据你的操作做出不同表情，还会定期提醒你完成目标。

![迪莫悬浮球演示](https://via.placeholder.com/400x200?text=Dimo+Demo) <!-- 你可以放一张动图或截图 -->

---

## ✨ 功能概览

| 功能 | 说明 |
|------|------|
| ✅ 每日打卡 | 记录坚持天数，每次打卡 +1 积分，连续 7 天额外 +1 积分 |
| 🎯 多旗帜系统 | 可同时立多个 Flag（运动、学习、健康等），完成一个解锁一个里程碑 |
| 🌟 愿望与兑换 | 用积分兑换“一杯奶茶”、“一顿饭”或“实现一个愿望” |
| 💬 PushPlus 提醒 | 配置个人 token 后，每晚 22:00 自动提醒未打卡用户 |
| 🗺️ 里程碑地图 | 每完成一个 Flag，地图上点亮一颗星星，路径逐渐完整 |
| 📅 日历打卡记录 | 查看每月打卡情况，打卡日有铅笔画风标记 |

---

## 🐾 迪莫悬浮球交互规则

| 你的操作 | 迪莫的反应 | 说明 |
|----------|------------|------|
| **单击** 头像 | 眨眼（wink） | 仅表情，不打开主窗口 |
| **双击** 头像 | 开心表情 + 打开主界面 | 打开打卡网页 |
| **按住并拖动** 头像 | 惊讶表情 | 只要按住超过 0.2 秒并移动，就会触发 |
| **闲置 10 秒** 不操作 | 打哈欠（犯困） | 任何操作（单击/双击/拖动）都会唤醒 |
| 呼吸、自动眨眼 | 一直持续 | 无需操作，永远在呼吸，每 4 秒自动眨眼 |

## 🚀 如何运行（给朋友看的）

### 1️⃣ 安装 Node.js

- 访问 [nodejs.org](https://nodejs.org/)，下载 LTS 版本并安装。
- 安装完成后，打开终端（Mac）或命令提示符（Win），输入 `node -v` 确认安装成功。
